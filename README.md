## Prerequisites

- Flutter 3.22.1
  - sdk: '>=3.4.1 <4.0.0'
  - Dart 3.4.1
- Android 5.0 - 14.0
  - compileSdkVersion = 34
  - minSdkVersion = 21
  - targetSdkVersion = 34
  - ndkVersion = "23.1.7779620"
  - com.android.tools.build:gradle:7.3.0

## Run

1. 下载包 `flutter pub get`
2. 查看可用设备 `flutter devices`
3. 运行 `flutter run -d [设备名称]`
